# PreCICE 

[PreCICE](https://precice.org/) is an open-source coupling library for partitioned multi-physics simulations. It couples existing programs/solvers capable of simulating a subpart of the complete physics involved in a simulation. 

These instructions cover installing: 
* [preCICE](https://precice.org/)
* [preCICE OpenFOAM adapter](https://precice.org/adapter-openfoam-overview.html) 
* [CalculiX](https://www.dhondt.de/) 
* [preCICE CalculiX adapter](https://precice.org/adapter-calculix-overview.html)
